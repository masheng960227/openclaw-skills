# 平面设计技能 (Graphic Design)

## 概述
平面设计是通过视觉元素传达信息和概念的艺术，包括排版、图像处理、色彩运用和版式设计。

## 核心原则

### 1. 对比 (Contrast)
```
作用:
- 吸引注意力
- 组织信息层次
- 创造视觉兴趣

实现方式:
- 大小对比
- 颜色对比
- 粗细对比
- 疏密对比
- 明暗对比
```

### 2. 重复 (Repetition)
```
作用:
- 建立统一性
- 增强识别度
- 创造节奏感

应用元素:
- 颜色重复
- 形状重复
- 字体重复
- 图形元素重复
- 间距重复
```

### 3. 对齐 (Alignment)
```
作用:
- 创造秩序感
- 建立视觉联系
- 提升专业度

对齐方式:
- 左对齐
- 右对齐
- 居中对齐
- 两端对齐
- 网格对齐
```

### 4. 亲密性 (Proximity)
```
作用:
- 组织相关信息
- 减少视觉混乱
- 引导阅读顺序

应用原则:
- 相关元素靠近
- 无关元素分离
- 分组清晰
- 留白适当
```

## 版式设计

### 1. 网格系统
```
常见网格:
- 单栏网格 (Single Column)
- 多栏网格 (Multi-Column)
- 模块网格 (Modular Grid)
- 层次网格 (Hierarchical Grid)

应用要点:
- 建立视觉秩序
- 保持灵活性
- 考虑内容优先级
- 适当打破规则
```

### 2. 留白艺术
```
留白作用:
- 突出主要内容
- 创造呼吸空间
- 提升品质感
- 引导视觉焦点

留白类型:
- 行间距 (Leading)
- 字间距 (Tracking)
- 段落间距
- 边距 (Margin)
- 内边距 (Padding)
```

### 3. 视觉流程
```
常见流程:
- Z 型布局 (Z-Pattern)
- F 型布局 (F-Pattern)
- 古腾堡图表 (Gutenberg Diagram)
- 中心辐射型

设计要点:
- 符合阅读习惯
- 引导视线移动
- 重点内容优先
- 自然流畅
```

## 字体设计

### 1. 字体分类
```
衬线体 (Serif):
- 传统、正式、优雅
- 适合：书籍、报纸、正式文档
- 示例：Times New Roman, Georgia, 宋体

无衬线体 (Sans-serif):
- 现代、简洁、友好
- 适合：网页、UI、海报
- 示例：Helvetica, Arial, 黑体

手写体 (Script):
- 个性、艺术、优雅
- 适合：标题、邀请函、品牌
- 示例：Brush Script, 行书

装饰体 (Display):
- 独特、吸引眼球
- 适合：标题、海报、包装
- 示例：各种创意字体
```

### 2. 字体搭配
```
搭配原则:
- 不超过 3 种字体
- 风格互补
- 用途区分
- 保持对比

经典搭配:
- 衬线标题 + 无衬线正文
- 粗体标题 + 细体正文
- 西文 + 中文搭配
```

### 3. 排版技巧
```
字号层次:
- 主标题：48px+
- 副标题：32px
- 小标题：24px
- 正文：16px
- 辅助文字：14px

行高设置:
- 标题：1.2-1.4
- 正文：1.5-1.8
- 大段文字：1.6-2.0

字距调整:
- 大标题：适当收紧
- 正文：标准字距
- 全大写：适当放宽
```

## 色彩设计

### 1. 色彩理论
```
色彩三要素:
- 色相 (Hue): 颜色的种类
- 饱和度 (Saturation): 颜色的纯度
- 明度 (Value): 颜色的明暗

色彩关系:
- 原色：红、黄、蓝
- 间色：橙、绿、紫
- 复色：三种以上颜色混合
```

### 2. 配色方案
```
单色配色 (Monochromatic):
- 同一色相不同明度/饱和度
- 和谐统一
- 适合：简约设计

类似色 (Analogous):
- 色相环相邻颜色
- 自然和谐
- 适合：自然主题

互补色 (Complementary):
- 色相环相对颜色
- 强烈对比
- 适合：吸引注意

分裂互补 (Split-Complementary):
- 主色 + 互补色两侧颜色
- 对比强烈但柔和
- 适合：平衡设计

三元色 (Triadic):
- 色相环等距三色
- 丰富活泼
- 适合：儿童、创意
```

### 3. 色彩心理学
```
红色:
- 情感：热情、危险、爱
- 应用：促销、警告、餐饮

蓝色:
- 情感：信任、专业、冷静
- 应用：科技、金融、医疗

绿色:
- 情感：自然、健康、成长
- 应用：环保、食品、金融

黄色:
- 情感：快乐、活力、警示
- 应用：儿童、餐饮、促销

紫色:
- 情感：高贵、神秘、创意
- 应用：美容、艺术、奢侈品

橙色:
- 情感：活力、友好、实惠
- 应用：食品、运动、促销

黑色:
- 情感：优雅、力量、神秘
- 应用：奢侈品、科技、时尚

白色:
- 情感：纯洁、简洁、空间
- 应用：医疗、科技、极简
```

## 图像处理

### 1. 图像选择
```
选择原则:
- 符合主题
- 高质量
- 版权清晰
- 情感共鸣

图像类型:
- 摄影图片
- 插画
- 图标
- 图案
- 抽象图形
```

### 2. 图像编辑
```
基础调整:
- 亮度/对比度
- 饱和度
- 色温
- 裁剪
- 旋转

高级处理:
- 抠图
- 合成
- 调色
- 特效
- 修复
```

### 3. 图像优化
```
Web 优化:
- 格式选择 (JPG/PNG/WebP/SVG)
- 压缩质量
- 尺寸适配
- 延迟加载

印刷优化:
- 分辨率 300dpi+
- CMYK 色彩模式
- 出血设置
- 字体转曲
```

## 印刷知识

### 1. 印刷基础
```

- RGB: 屏幕显示
- CMYK: 印刷输出
- Pantone: 专色印刷

分辨率:
- 屏幕：72dpi
- 印刷：300dpi
- 大幅印刷：150dpi

### 2. 印刷工艺
```
常见工艺:
- 覆膜 (Glossy/Matte)
- 烫金/烫银
- UV 局部上光
- 凹凸压纹
- 模切成型
- 骑马钉/胶装
