# Humanizer 技能 (AI 文本人性化)

## 概述
将 AI 生成的文本转换为更自然、更像人类书写的风格，降低 AI 检测率，提升可读性。

## 核心功能

### 1. 风格转换
- **正式 → 自然**: 去除过于正式的措辞
- **机械 → 流畅**: 增加过渡词和连接语
- **单调 → 生动**: 添加情感色彩和个人观点

### 2. 人性化技巧

#### 句式变化
```
❌ AI 风格: 首先，其次，最后，综上所述
✅ 人类风格: 一开始，然后，还有一个重点是，总的来说
```

#### 情感注入
```
❌ AI 风格: 这个功能可以提高效率
✅ 人类风格: 说实话，这个功能真的能省不少时间
```

#### 个人化表达
```
❌ AI 风格: 根据研究表明
✅ 人类风格: 我之前遇到过类似情况，发现...
```

### 3. 检测规避

| AI 特征 | 人类化替换 |
|---------|------------|
| 过于完美的结构 | 适当打破段落平衡 |
| 重复的句式模式 | 混合长短句 |
| 缺少个人观点 | 加入主观评价 |
| 过度使用连接词 | 自然过渡 |
| 完美语法 | 偶尔口语化表达 |

## 使用方法

### 基础人性化
```bash
# 输入 AI 生成的文本
input = """
首先，人工智能技术的发展为各行各业带来了深远的影响。
其次，自动化流程可以显著提高工作效率。
最后，建议企业积极拥抱这一技术变革。
"""

# 人性化输出
output = """
说实话，AI 技术这几年真的改变了很多行业。
我自己用过一些自动化工具，效率提升特别明显。
所以我的建议是，别犹豫，早点尝试准没错。
"""
```

### 程度控制
```python
# 人性化程度：0-100
# 0 = 保持原样
# 50 = 适度优化
# 100 = 完全口语化

humanize(text, level=50)  # 平衡模式
humanize(text, level=80)  # 高度口语化
humanize(text, level=30)  # 轻度优化
```

## 常见场景

### 场景 1: 降低 AI 检测率
```
目标：通过 Originality.ai 等检测工具
策略：
  1. 打乱段落结构
  2. 加入个人经历
  3. 使用不规则句式
  4. 添加情感词汇
```

### 场景 2: 提升可读性
```
目标：让读者感觉更亲切
策略：
  1. 使用第二人称"你"
  2. 添加问句互动
  3. 使用比喻和类比
  4. 加入幽默元素
```

### 场景 3: 品牌声音统一
```
目标：符合品牌调性
策略：
  1. 定义品牌人格（专业/友好/幽默）
  2. 创建词汇表（禁用/推荐用词）
  3. 建立句式模板
  4. 保持一致的情感基调
```

## 技术实现

### 核心算法
```python
def humanize(text, level=50):
    # 1. 分析原文本特征
    features = analyze_style(text)
    
    # 2. 识别 AI 痕迹
    ai_markers = detect_ai_patterns(text)
    
    # 3. 应用人性化转换
    if level >= 50:
        text = vary_sentence_structure(text)
        text = add_personal_touch(text)
        text = replace_formal_words(text)
    
    if level >= 70:
        text = add_colloquialisms(text)
        text = introduce_minor_imperfections(text)
    
    if level >= 90:
        text = add_anecdotes(text)
        text = inject_emotion(text)
    
    return text
```

## 质量评估

### 评估指标
| 指标 | 描述 | 目标值 |
|------|------|--------|
| 自然度 | 读起来像人类的程度 | > 80% |
| 流畅度 | 语句通顺程度 | > 90% |
| 情感丰富度 | 情感表达程度 | > 60% |
| AI 检测通过率 | 通过检测工具的概率 | > 85% |
| 信息保留率 | 原意保留程度 | > 95% |

### 人工评估
```
评分标准（1-5 分）：
5 = 完全像人类写作，无法分辨
4 = 大部分像人类，偶有 AI 痕迹
3 = 一半一半，需要进一步优化
2 = 明显 AI 风格，需要大量修改
1 = 完全机械化，毫无人类特征
```

## 注意事项 ⚠️

### 不要过度人性化
```
❌ 过度：哎呀妈呀这个真的太好用了我跟你说！！！
✅ 适度：说实话，这个真的挺好用的。
```

### 保持专业性
```
❌ 不合适：这个数据分析简直弱爆了
✅ 合适：这个数据分析还有优化空间
```

### 考虑受众
```
正式场合 → 轻度人性化 (30-50%)
日常沟通 → 中度人性化 (50-70%)
社交媒体 → 高度人性化 (70-90%)
```
